# 人生清单小程序 - 简化版

## 🎯 项目概述

这是一个简化的人生清单小程序，用户选择模板后直接勾选预设选项，然后生成精美的图片分享。

## ✨ 核心特性

- **4种精美模板**，每个模板都有预设的清单选项
- **简单直接**的用户流程：选择模板 → 勾选选项 → 生成图片
- **Canvas图片生成**，保存到相册分享
- **响应式设计**，支持不同屏幕尺寸

## 📱 使用流程

1. **选择模板** - 在首页选择喜欢的模板样式
2. **勾选选项** - 在模板页面勾选已完成的项目
3. **生成图片** - 点击按钮生成精美图片并保存

## 🎨 模板介绍

### Template 1: 简约清新
- 蓝色主题，带头像装饰
- 10个人生成长相关选项
- 底部感言区域

### Template 2: 正式简洁  
- 绿色主题，分隔线设计
- 8个人生目标相关选项
- 简洁专业风格

### Template 3: 复古温馨
- 复古纹理背景，装饰元素
- 10个生活愿望相关选项
- 温馨复古风格

### Template 4: 标签云
- 彩色标签云布局
- 100+个电影清单选项
- 适合大量选项展示

## 🚀 快速开始

### 1. 安装依赖

```bash
cd checklist-app
npm install
```

### 2. 运行开发

```bash
npm run dev:mp-weixin
```

### 3. 微信开发者工具

1. 打开微信开发者工具
2. 导入项目：`dist/dev/mp-weixin`
3. 填入你的AppID（在 `src/manifest.json` 中）
4. 开始调试

### 4. 构建发布

```bash
npm run build:mp-weixin
```

## 🔧 自定义模板

每个模板都是独立的配置文件，包含：

```javascript
export default {
  id: 'template1',
  name: '模板名称',
  title: '清单标题',
  description: '模板描述',
  
  // 预设选项
  presetItems: [
    '选项1',
    '选项2',
    // ...
  ],
  
  // 样式配置
  styles: {
    backgroundColor: '#f5f1e8',
    titleColor: '#333333',
    // ...
  },
  
  // Canvas渲染配置
  canvas: {
    width: 750,
    height: 1200,
    // ...
  }
}
```

## 📝 添加新模板

1. 在 `src/templates/` 创建新的模板文件
2. 在页面中引入新模板
3. 添加对应的Canvas渲染逻辑

## 🎯 技术特点

- **配置驱动**：每个模板都是独立的配置文件
- **组件化**：清晰的页面和组件结构  
- **Canvas渲染**：支持多种布局的图片生成
- **响应式**：适配不同设备和屏幕
